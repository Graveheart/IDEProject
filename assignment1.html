Interactive Data Exploration Warming up

  1. Music Timeline

    Pros:

    • User friendly
    • Provides search option
    • Albums for selected area get on top
    • Shows the seven most popular albums

    Cons:

    • Loading div is shown even after loading is done
    • Music genre is not shown after searching
    • Colors appear to be random
    • Repeating colors for different items makes telling them apart hard
    • Not that much user customization

  2. Track National Unemployment

    Pros:

    • Provides detailed information
    • Provides legend for colors
    • Hovering over a sector shows a dot on that sector for previous/next months
    • Clicking on a month gives you a detailed view table for that month
    • Has a sorting option

    Cons:

    • No search or filtering makes it a hard to find a particular sector you are looking for
    • Sectors are too general

Project: Browser History

  1. What type of data would you like to visualize, and where would you get access to this data?

    Browser history from Google Chrome history, can be either accessed locally through a Google Chrome extension or through a csv file with real-life data.

  2. What story would the combination of text and visualization tell?

    Google Chrome history is far from perfect and finding something you are looking for can be very hard at times. I would provide a solution for a better way of visualizing browser history with some custom features.

  3. Who is the intended audience?

    Everyone is browsing excessively nowadays and creating this as a web application means it can be used by abosutely everyone. Another alternative is to restrict it a bit and address only Google Chrome as it is the most widely used browser by creating an extension.
